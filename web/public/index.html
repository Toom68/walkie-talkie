Push-to-Talk

Press and hold to transmit. Release to listen. Just like a real walkie-talkie — no open mics, no background noise.

🚫

No Crosstalk

Only one person can transmit at a time per channel. The server enforces strict floor control — no talking over each other.

📡

100 Channels

Choose from 100 independent channels. Organize teams, groups, or events on separate frequencies.

👥

Multi-User

Unlimited users can join the same channel. Everyone hears the transmission, one person speaks at a time.

⚡

Low Latency

WebRTC peer-to-peer audio with sub-50ms latency. Real-time communication that feels instant.

🔒

Queue System

If someone is already transmitting, you're placed in queue. Your turn comes automatically when the channel clears.

How It Works

Simple as a real walkie-talkie

1

Set Your Callsign

Choose a unique name so others can identify you on the channel.

2

Join a Channel

Select any channel from 1-100. Anyone on the same channel can hear you.

3

Push to Talk

Press and hold the big PTT button to transmit. Release when done.

4

Listen

When someone else is transmitting, you'll hear them automatically.

📱 Installation Guide

  • Android: Download the APK file above. You may need to enable "Install from Unknown Sources" in your device settings. Open the file and tap Install.
  • iOS: Download the IPA file. You'll need to sideload it using AltStore, Sideloadly, or a similar tool. Connect your device and follow the tool's instructions.
  • Permissions: Grant microphone access when prompted. The app only transmits audio when you press the PTT button.
  • Server: The app connects to our signaling server automatically. No additional setup required.
    Server: walkie-talkie-server.onrender.com
  • Build from Source: Clone the repo and build with React Native CLI.
    git clone <repo-url> && cd mobile && npm install && npx react-native run-android
